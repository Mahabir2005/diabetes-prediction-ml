# Diabetes Prediction using Machine Learning

## 📌 Overview
This project predicts whether a person is diabetic or not using machine learning techniques based on medical attributes.

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Model training using ML algorithms
- Accuracy evaluation
- User input-based prediction

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 📊 Dataset
PIMA Diabetes Dataset

## 🤖 Model Used
Logistic Regression

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the script:
   python src/predict.py
